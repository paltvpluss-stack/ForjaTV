# 📺 FORJA TV — تطبيق أندرويد للبث المباشر

تطبيق أندرويد يشغّل قنوات FORJA TV مباشرة بجودة HD.

---

## 🚀 طريقة بناء APK (بدون Android Studio)

### الخطوة 1: إنشاء حساب GitHub
اذهب إلى [github.com](https://github.com) وسجّل حساباً مجانياً.

### الخطوة 2: إنشاء Repository جديد
- اضغط **New repository**
- اسمه: `ForjaTV`
- اجعله **Public**
- اضغط **Create repository**

### الخطوة 3: رفع ملفات المشروع
- اضغط **uploading an existing file**
- ارفع جميع ملفات المجلد هذا مع الحفاظ على بنية المجلدات

### الخطوة 4: GitHub Actions يبني APK تلقائياً
- اذهب إلى تبويب **Actions** في المستودع
- ستجد build يعمل تلقائياً
- انتظر ~5 دقائق

### الخطوة 5: تنزيل APK
- بعد انتهاء البناء، اذهب إلى **Releases** أو **Actions → آخر build → Artifacts**
- نزّل ملف `FORJA-TV-v1.0-debug.apk`

---

## 📱 تثبيت APK على الهاتف

1. انقل ملف APK إلى هاتفك
2. في الهاتف: الإعدادات ← الأمان ← السماح بمصادر غير معروفة
3. افتح ملف APK واضغط تثبيت

---

## ✅ مميزات التطبيق

- يشغّل بثوث IPTV مباشرة بدون أي خادم وسيط
- لا توجد مشكلة CORS (ExoPlayer يتجاوزها تلقائياً)
- واجهة احترافية داكنة
- يعمل على أندرويد 5.0 فأحدث
- يمكن مشاركته مع أي شخص
